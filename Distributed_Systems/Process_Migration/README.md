Process-Migration
=================

Lab 1 of 15-440/640

ProcessMigration.

Compile and Build
-----------
`cd` into the folder `Process-Migration` and build it using
```
make clean
make
```

Execution
-----------
Run the program using
```
java -cp . ProcessMigration -p PortNumber
```

Commands
-----------
You can use `ls`, `launch`, `migrate`, `kill`, and `quit`. Use `help` to find out about these commands.
